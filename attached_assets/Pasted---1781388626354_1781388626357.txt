Создай веб-приложение "вайб-кодинг сервис" — пользователь описывает 
приложение словами, ИИ генерирует код, код запускается в песочнице, 
пользователь видит живое превью и правит результат текстом.

СТЕК:
- Next.js (App Router, TypeScript)
- Tailwind CSS для стилей
- Postgres (Replit DB или Neon) для проектов и истории
- Anthropic Claude API для генерации кода
- E2B SDK для запуска кода в песочнице

ЭКРАН (один основной):
Раздели на 3 колонки:
1. Левая — чат: поле ввода задачи + история сообщений юзер/ИИ
2. Центр — список сгенерированных файлов с подсветкой кода
3. Правая — iframe с живым превью + кнопка "Обновить"
Сверху — название проекта и кнопка "Новый проект".

ЛОГИКА (главный цикл):
1. Юзер пишет задачу в чат
2. Бэкенд собирает: системный промпт + текущие файлы проекта + новое сообщение
3. Вызов Claude API (model: claude-sonnet-4-6), ответ строго в JSON:
   { "files": [{ "path": "...", "content": "..." }], "message": "что сделал" }
4. Бэкенд парсит JSON, сохраняет файлы в БД (привязка к проекту)
5. Файлы записываются в E2B sandbox, запускается dev-сервер
6. URL превью из E2B возвращается на фронт в iframe
7. Юзер пишет правку — цикл повторяется с обновлённым контекстом

СИСТЕМНЫЙ ПРОМПТ ДЛЯ CLAUDE (вшей в бэкенд):
"Ты генерируешь полный рабочий код фронтенд-приложения. Всегда возвращай 
ВСЕ файлы проекта целиком (не диффы). Отвечай ТОЛЬКО валидным JSON без 
markdown-обёрток, в формате: {files: [{path, content}], message}. 
Используй простой стек (HTML/CSS/JS или React). Код должен запускаться 
сразу без ручной настройки."

БД (схемы):
- projects: id, name, created_at
- files: id, project_id, path, content, updated_at
- messages: id, project_id, role, content, created_at

API-КЛЮЧИ (через Replit Secrets, не хардкодь):
- ANTHROPIC_API_KEY
- E2B_API_KEY

ВАЖНО:
- Ключи только из переменных окружения / Secrets
- Обрабатывай ошибки парсинга JSON от Claude (try/catch + повтор)
- Начни с рабочего каркаса: сначала цикл "промпт → код → превью", 
  биллинг/авторизацию/деплой НЕ делай
- Каждый проект = отдельный E2B sandbox

в Secrets перед запуском.
